# webpack-starter-scss-js
Work in progress, feel free to participate


## Features
- Webpack configuration
- JS (babel)
- SCSS (autoprefixer, cssnano)
- Browser-sync 
## Requirements
- Node.js

## Installation 
    git clone https://github.com/nasedkinpv/webpack-starter-scss-js.git
    npm install
    
## Start server
    npm run dev
